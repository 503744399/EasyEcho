# EasyEcho
EasyEcho 用于echo时的一键转义


Usage of EasyEcho.exe:
  -t string
        文件路径，默认为空 (default "target")


例子：
直接echo有需要转义的字符串时无法正常echo

![image](https://user-images.githubusercontent.com/85235087/200552323-84affca0-37b2-4ded-bec7-89313ac689c3.png)
